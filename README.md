# keypress

Various code snippets to respond to keypresses.

See:

* [index.html](index.html) - Detect Enter being pressed anywhere on the page, consumes the event (so it cannot affect other elements), and ignores key repeats if the button is held.
* [blocking_example.py](blocking_example.py) - Blocking terminal input (requires Enter key, subject to key repeat)
* [minimum_interval.py](minimum_interval.py) - Blocking terminal input (requires Enter key, ignores key repeat)
* [key_reader.py](key_reader.py) - Non-blocking terminal keypress reader
